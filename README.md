# القافلة الآمنة — بوابة العملاء (نموذج تفاعلي / MVP)

نموذج تصميمي تفاعلي (interactive prototype) لبوابة عملاء شركة **القافلة الآمنة** للشحن من الصين إلى ليبيا.
واجهات عربية RTL للموبايل، مصمّمة على هوية الشركة (الأزرق الملكي `#1f5aa8`).

> ⚠️ هذا نموذج للعرض فقط — الأزرار والحقول للتوضيح البصري ولا تنفّذ عمليات فعلية.

## الشاشات (8 من MVP)
1. تسجيل الدخول
2. إنشاء حساب جديد
3. تنبيه بشحنة قادمة (Pre-Alert)
4. طلباتي وتنبيهاتي
5. مدفوعاتي وإيصالاتي
6. كشف الحساب الشامل
7. الأمان وكلمة المرور
8. الأسئلة الشائعة

## التشغيل محلياً
موقع ثابت (static) بدون أي بناء — افتح `index.html` مباشرة، أو:

```bash
npx serve .
```

## التقنية
- HTML + Tailwind (CDN) + خط Noto Sans Arabic
- صُمّمت الشاشات في Google Stitch ثم صُدّرت كـ HTML ثابت
- الصفحة الرئيسية `index.html` تعرض الشاشات داخل إطارات موبايل
